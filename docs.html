Cerebellum Documentation

Welcome to the official Cerebellum documentation. This guide will help you understand how to use our platform to discover, integrate, and build robotic world models.

What is Cerebellum?

Cerebellum is the marketplace for open source robot developers to share, monetize, and distribute world models for any hardware. Our platform provides a standardized way to package, distribute, and integrate sophisticated AI models into robotics applications.

World models allow robots to understand and predict their environment, enabling more intelligent behavior and decision-making. Cerebellum makes these models accessible to developers of all levels.

This documentation assumes basic familiarity with robotics concepts and programming. If you're new to these topics, we recommend starting with our beginners tutorials.

Quick Start Guide

1. Install the Cerebellum SDK

Install our SDK using pip:

pip install cerebellum-sdk

2. Initialize with your API key

Authenticate with your API key:

import cerebellum

cerebellum.init(api_key="your_api_key_here")

3. Load a model and make predictions

Load a model and use it to make predictions:

import cerebellum
import numpy as np

# Initialize with your API key
cerebellum.init(api_key="your_api_key_here")

# Load a world model
model = cerebellum.load("domestic-navigator")

# Create sample sensor data
sensor_data = {
    "lidar": np.random.rand(360) * 5,  # 360° lidar scan with 5m max range
    "position": [0, 0],
    "orientation": 0
}

# Get predictions
prediction = model.predict(sensor_data)
print(prediction)

Congratulations! You've successfully loaded a Cerebellum world model and made your first prediction. The prediction result contains environment understanding and recommended actions for your robot.

Installation

System Requirements

  • Python 3.7 or higher
  • Linux, macOS, or Windows operating system
  • Internet connection for model downloads and API access
  • GPU recommended for real-time inference (CUDA compatible)

Installation Methods

Using pip (Recommended)

pip install cerebellum-sdk

Using conda

conda install -c cerebellum cerebellum-sdk

From source

git clone https://github.com/cerebellum/cerebellum-sdk.git
cd cerebellum-sdk
pip install -e .
