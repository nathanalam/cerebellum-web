Integration Tools

CLI Tool

Our command-line interface makes packaging and publishing world models simple and efficient.

$ npm install -g cerebellum-cli
View CLI Documentation →

SDK Integration

Integrate Cerebellum models into your robotics projects with our simple SDK.

import cerebellum

model = cerebellum.load("domestic-navigator")
results = model.predict(sensor_data)
View SDK Documentation →

API Access

Access models via REST API for easy integration with any platform or language.

curl -X POST https://api.cerebellum.ai/models/predict \
-H "Authorization: Bearer $API_KEY" \
-d '{"model_id": "domestic-navigator", "data": {...}}'
